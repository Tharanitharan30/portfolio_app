# Tharani Portfolio

Flutter foundation for a modern personal portfolio application using Material 3.

## What is included

- Feature-first folder structure with shared `app` and `core` layers
- Centralized Material 3 light and dark theme setup
- Route generation and bottom navigation foundation
- Reusable placeholder shell for future portfolio sections
- Asset directories prepared for images, icons, animations, and data
- Stronger linting for clean, scalable code


## Notes

- Real portfolio content has intentionally not been implemented yet.
- Add future state management, repositories, and models inside each feature as the app grows.
